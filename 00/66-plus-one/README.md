# [66. Plus One](https://leetcode.com/problems/plus-one/)

Increment the **large integer** by one and return the _resulting array of digits_.

### Example 1:

```
Input: digits = [1,2,3]
Output: [1,2,4]
```

### Example 2:

```
Input: digits = [9,9]
Output: [1,0,0]
```

### Constraints:

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` does not contain any leading `0`'s.
